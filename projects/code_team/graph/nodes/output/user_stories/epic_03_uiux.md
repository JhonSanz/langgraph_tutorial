# Epic EPIC_03_UIUX: Experiencia de usuario e interfaz

## Business Objective
Ofrecer una UI intuitiva y moderna, optimizada para productividad y feedback inmediato.

## User Stories
- user_story_04 (3 pts): Filtros y búsqueda easy/wow
- user_story_05 (3 pts): UI polish y métricas UX

## Success Metrics
- NPS >8 en feedback usuarios
- <300ms render pantalla principal

## Riesgos
- Usuarios frustrados (UX mala) – Medio – Test US+feedback, releases incrementales
